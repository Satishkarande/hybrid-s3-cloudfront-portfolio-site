Secure Static Website Hosting (S3 + CloudFront + OAI)

This website is hosted on a private S3 bucket and served securely using Amazon CloudFront with Origin Access Identity (OAI). The architecture follows AWS best practices for global delivery, caching, and security.

Status: Successfully deployed and running on CloudFront.

About This Deployment

• S3 bucket is fully private (block public access ON)
• CloudFront serves the content securely over HTTPS
• Bucket access allowed only through OAI
• Website files deployed using AWS CLI (s3 sync)
• Cache invalidation handled via CloudFront invalidation API

Built on AWS | S3 • CloudFront • OAI • CLI
